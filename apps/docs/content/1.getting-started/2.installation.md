evlog 支持 Nuxt、Next.js、SvelteKit、Nitro、NestJS 以及任何 TypeScript 服务端框架。

## 安装

::code-group
```bash [pnpm]
pnpm add evlog
```
```bash [npm]
npm install evlog
```
```bash [yarn]
yarn add evlog
```
```bash [bun]
bun add evlog
```
::

### 使用 Agent 技能

如果你使用了 AI 助手（如 Claude Code、Cursor 等），安装 evlog 技能以获得引导式设置和代码审查：

```bash [Terminal]
npx skills add https://www.evlog.dev
```

你的 AI 助手随后可以帮助你设置 evlog、审查你的日志模式，并将现有代码迁移到宽事件。详情参见 [Agent Skills](/getting-started/agent-skills)。

## TypeScript 配置

evlog 附带完整的 TypeScript 类型定义，无需额外配置。

::callout{icon="i-lucide-check" color="success"}
evlog 需要 TypeScript 5.0 或更高版本以获得最佳的类型推断。
::
